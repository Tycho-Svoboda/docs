## tensorpool.realize()
This function is called on a machine learning model. 

It sends the computation graph of the model to the TensorPool server. We train it for you using the best GPU for the job.

## tensorpool.get_comp_graph()
This function is called on a machine learning model.

It fetches the computation graph of a model.
